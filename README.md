# 🏦 Loan Prediction System using Logistic Regression

## 📌 Project Overview
This is an end-to-end Machine Learning project that predicts whether a loan application will be approved or rejected based on user input features. The model is built using **Logistic Regression** and deployed as a web application using **Flask**.

## 🚀 Features
* Predicts loan approval status (Approved / Rejected)
* Displays probability score of approval
* User-friendly web interface
* Real-time prediction using trained ML model
* End-to-end pipeline (data → model → deployment)

## 🧠 Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Feature Scaling (`StandardScaler`)
5. Model Training (`Logistic Regression`)
6. Model Evaluation
7. Model Deployment using Flask

## 🛠️ Tech Stack
* **Python**
* **Flask**
* **Scikit-learn**
* **NumPy**
* **Joblib**
* **HTML/CSS**
